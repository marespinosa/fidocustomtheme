== Description ==
Tri is a lightweight yet powerful theme built for smart businesses. Tri is SEO optimized to help you achieve the best possible search engine results. With the Tri enhancer add-on, it is capable of achieving many unique designs and layouts.

The Tri theme has been used by businesses in over 45 different industries. Uniquely crafted for hundreds of big and small businesses around the world. The industries include: Health, Construction, Agriculture, Finance and Technology.
This theme has helped generate tens of millions of dollars for these businesses online.

== Frequently Asked Questions ==

= How do I customize Tri?

Modify the appearance through the WordPress Customizer. All of the styles are controlled through CSS. It is recommended that you use SCSS to properly import the Tri theme css into a child theme.

= Where can I find the theme documentation? =

https://tri.net/documentation is what you are looking for. You can also preview a demo of the theme at https://tri.net/

= Does this theme support any plugins? =

Tri has been built alongside a handful of supplementary plugins. Tri Enhancer is the number one recommended plugin for Tri. It adds shortcodes, widgets and other lightweight yet powerful goodness to the theme.
